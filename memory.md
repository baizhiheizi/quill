## Notes

- `bin/rubocop` does NOT lint ERB or YAML. `bin/rails zeitwerk:check` is the local CI load-path check.
- **Stimulus dead-code detection**: scan `data-controller="xxx"`, `data-controller='xxx'`, `controller: 'xxx'`. "Dead" only if (a) no view references it AND (b) the controller file does not exist.
- **Empty controller stub detection**: a Stimulus controller with only a `connect()` that does `console.log(...)` is a migration stub.
- **Dead Rails action detection**: for any `def <verb>` on a controller, confirm the verb is exposed by the matching route AND a view file exists.
- **Admin filter typo pattern**: `where(col_a: params[:col_b], col_b: params[:col_b])`.
- **Pre-existing CI-clean test errors**: `test/controllers/articles_controller_test.rb` (3 errors, asset pipeline) and `test/controllers/pre_orders_controller_test.rb` (2+ errors, SSL to Mixpay). CI stubs/supplies them.
- **`remote_image_tag`** (`ef58c599`): `nil` for nil/blank or non-absolute-URL; otherwise `image_tag`. `lazy: true` → `loading: "lazy"`.
- **`User#available_articles` perf**: push `.uniq` to SQL via `.or.or.distinct`.
- **`includes(:currency).sum(...)` antipattern**: use `joins(:currency)`. **Closed in `Article#author_revenue_usd` / `#reader_revenue_usd`**.
- **`HomeController#active_authors` perf**: PR #1735 uses `NOT IN (SELECT ...)`.
- **`setTimeout(fn(), ms)` antipattern**: invokes `fn()` synchronously. Fixed in `cbd10abd`.
- **`TokenAsset`** (PORO at `app/models/token_asset.rb`): Balance = `token["balance"].to_r / 10**token["decimals"].to_i` (round 8 dp), or 0 if `token.blank?`. Delegates 9 accessors to currency. `attr_reader :contract` declared but `@contract` never assigned — returns `nil`.
- **`Currency#save` raises in test env** because `before_validation :set_defaults` calls `QuillBot.api.asset(asset_id)`. Tests use `Currency.new(price_usd: ...)` in memory.

- **`CreateTagService` subtle behavior**: `tag_names.blank?` short-circuits only for `nil` and `[]`. Non-empty array of blanks gets `compact_blank!` cleanup.
