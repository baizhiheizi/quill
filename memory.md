Use `re.findall(r'data-controller=["\']([^"\']+)["\']', content)`. An identifier is "dead" only if (a) no view references it AND (b) the controller file does not exist. - `Action` extends `ActiveRecord::Base` directly (not `ApplicationRecord`), polymorphic `target`/`user` with `optional: true`, has `after_create :notify_target` + `before_destroy :destroy_notifications`.
