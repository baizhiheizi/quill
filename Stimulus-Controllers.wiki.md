## Catalog

Quill currently registers **40 Stimulus controllers** in [`app/javascript/controllers/index.js`](https://github.com/baizhiheizi/quill/blob/main/app/javascript/controllers/index.js) — auto-generated by `./bin/rails stimulus:manifest:update`. Each kebab-case identifier maps to `app/javascript/controllers/<snake_case>_controller.js` and mounts via `data-controller="<id>"`.

Seven controllers previously listed here (`autosave`, `modal`, `reload`, `scroll-to`, `select-menu`, `switch-locale`, `toast`) were removed in [PR #1683](https://github.com/baizhiheizi/quill/pull/1683) because no view or test consumed them.

### Pagination and scrolling

| Identifier | Purpose |
|---|---|

## Patterns to know

**Debounce via `underscore`.** Wrap the *function*, not its return value — `debounce(classList.add, 1000)` invokes `classList.add` immediately because the return value is `undefined`. The `floating` controller header documents a real bug fixed by this discipline.

**Lifecycle cleanup.** `disconnect()` must release every listener and timer the controller added so Turbo reconnects don't accumulate. Templates: `floating` (stores the bound scroll handler in `this.boundOnScroll`, removes it by reference) and `infinite-scroll` (calls `observer.disconnect()` and nulls the reference). `flash` clears its `this.timeout` in both `hide()` and `disconnect()`.

**Turbo-Stream fetches.** Controllers that mutate server state without a full reload import `get`/`put` from `@rails/request.js` and pass `responseKind: "turbo-stream"`. Examples: `infinite-scroll`, `load-more`, `turbo`, `search`. Pair with `contentType: "application/json"` when the endpoint expects JSON.

**`useTransition` from `stimulus-use`.** `flash` uses `useTransition(this)` to get `enter()` / `leave()` helpers, then awaits `leave()` before removing the DOM node.

**Theme reload hook.** `darkmode` calls `reloadTheme()` from `mixin-messenger-utils` after each toggle so the Mixin messenger iframe stays in sync.

**Dead-code hygiene.** Removing a controller means deleting the file, its `import` + `application.register(...)` pair in `index.js`, and the matching row in `docs/reference/stimulus-controllers.md`. PR #1683 demonstrated the recipe (zero-consumer grep over `app/views/` and `test/` before deletion).
