## Gotchas

- **`Announcement#preview` returns the result of `perform_later`**, not the message hash. Capture API call payload in a closure; assert on what was passed to `QuillBot.api.plain_text/plain_post`.
- **`AdminNotificationService`** calls `QuillBot.api.plain_text(conversation_id:, data:)` (no recipient_id); `Announcement#deliver_as_text` calls `(conversation_id:, recipient_id:, data:)`. Stubs must make recipient_id optional.
- **`deliver_as_text`** batches via `in_groups_of(100, false)`; **`deliver_as_post`** is one job per message. Pin both separately.
- **`User.pluck(:mixin_uuid)` includes nils**. For "no users" tests, `User.delete_all` is cleaner than `User.update_all(mixin_uuid: nil)`.
- **`Rails.application.credentials.dig` returns nil in test.** Override with a `FakeCredentials` Struct that responds to `.dig(*keys)`, restore in `ensure`. Pattern is local for now.
- **`User` validation requires `uid`** — set `uid: SecureRandom.hex(8)` when creating users in-test.
- **`assert_enqueued_jobs N, only: JobClass`** is canonical. `enqueued_jobs.first[:job]` returns the class.
