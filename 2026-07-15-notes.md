## Gotchas (new this run)

- **`store_accessor` accepts arbitrary extra JSON keys** ‚Äî the four declared accessors are conveniences, not a schema. `raw["extra_field"]` round-trips fine and `snapshot.title` reads back correctly. Tested.
- **`set_defaults` does not guard against an explicitly provided raw** ‚Äî pre-supplying `raw:` in `create!` is silently clobbered. The test pins this so a future `return if raw.present?` guard is a deliberate decision, not a regression.
- **`fresh?` uses a fresh query**, not a cached count ‚Äî destroyed later snapshots correctly flip the answer back. Tested.
